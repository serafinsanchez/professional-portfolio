# Frontend Engineer at Vercel
*June 2020 - December 2021*

As a Frontend Engineer at Vercel, I had the opportunity to work on cutting-edge web technologies and contribute to the developer experience platform that powers millions of websites.

## What I Did

- **Dashboard Development**: Built and maintained the Vercel dashboard using React and Next.js
- **Developer Experience**: Improved deployment workflows and created intuitive user interfaces
- **Performance**: Optimized real-time analytics and deployment status updates
- **Open Source**: Contributed to Next.js and other open-source projects

## Key Projects

### Analytics Dashboard Redesign
Led the redesign of the analytics dashboard, implementing:
- Real-time data visualization with custom charts
- Responsive design for mobile and desktop
- Advanced filtering and search capabilities

### Deployment Pipeline UI
Created an intuitive interface for deployment pipelines:
- Real-time build logs and status updates
- Interactive deployment history
- One-click rollback functionality

## Technologies & Tools

- Next.js, React, TypeScript
- SWR for data fetching
- Tailwind CSS for styling
- Vercel Analytics and Edge Functions
- GitHub integration and CI/CD

Working at Vercel exposed me to the latest in web development technology and taught me the importance of developer experience. The collaborative environment and focus on innovation were incredible learning experiences.
