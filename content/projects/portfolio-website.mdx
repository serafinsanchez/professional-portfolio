# Developer Portfolio Website

A modern, responsive portfolio website built with Next.js 15, TypeScript, and MDX for easy content management.

## Project Overview

This portfolio showcases my work, skills, and experience in a clean, professional design. Built with performance and accessibility in mind, it features smooth animations and a great user experience.

## Key Features

- **MDX Content Management** - Easy to update content without touching code
- **Dark/Light Theme** - Automatic theme switching with system preference
- **Responsive Design** - Optimized for all devices
- **Performance Optimized** - 90+ Lighthouse scores
- **Contact Form** - Integrated with Resend API

## Tech Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Content**: MDX with @next/mdx
- **Animations**: Motion.dev
- **Deployment**: Vercel

## Development Process

The project started with a comprehensive PRD to define requirements and user experience goals. I focused on creating a content-first architecture that would be easy to maintain and extend.

## Results

- Achieved 95+ Lighthouse scores across all metrics
- Fast loading times (< 2s on 3G)
- Fully accessible (WCAG AA compliant)
- Easy content updates through MDX
